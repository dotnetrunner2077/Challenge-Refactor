# Challenge-Refactor

Repositorio que resuelve la siguiente consigna de refactorización de código, desarrollado con **.NET 8 / C#**.

---

## 📋 Consigna

> *¿Qué pasa si debemos soportar un nuevo idioma para los reportes, o agregar más formas geométricas?*

El punto de partida es una clase monolítica con lógica condicional para calcular áreas, perímetros y generar reportes en múltiples idiomas. El objetivo es refactorizarla para que el sistema sea fácilmente extensible ante nuevos requerimientos.

---
